Diwali Sales Analysis (EDA)

# Project Overview

- This project focuses on Exploratory Data Analysis (EDA) of a Diwali sales dataset using Python.
- The goal is to analyze customer purchasing behavior and extract meaningful insights using statistical methods and visualizations.

This project was completed as part of a Data Analyst Internship.


# Objective

- Understand customer demographics
- Analyze sales patterns based on gender and age groups
- Identify key contributors to revenue


Gain insights to support business decision-making



# Tools & Technologies Used

Python
> Pandas – data manipulation
> NumPy – numerical operations


Matplotlib & Seaborn – data visualization
Jupyter Notebook



# Dataset

- Diwali Sales Data (CSV file)


# Contains information about:
- User ID
- Customer Name
- Gender


- Age & Age Group

- State
-Occupation
- Product Category
- Orders
- Amount (Sales Value)



# Analysis Performed
1️. Gender Distribution

- Visualized the number of male and female customers using count plots.
- Observation: Female customers are higher in number compared to male customers.

  

2️. Total Sales Amount by Gender

- Grouped data by gender and calculated total sales amount.
- Observation: Female customers contribute more to overall sales.
  

3️. Age Group vs Gender Analysis

- Used count plots to analyze purchasing behavior across age groups.
- Observation: Customers in the 26–35 age group, especially females, made the most purchases.

  

4️. State-wise Orders Analysis

- Identified top states contributing to maximum orders.
- Visualized using bar plots.



- Key Insights

- Female customers dominate both purchase count and sales amount
- Age group 26–35 years is the most active buyer segment
- Certain states contribute significantly to total orders


Younger and middle-aged customers are more engaged in shopping during Diwali
